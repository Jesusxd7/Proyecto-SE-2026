## INTRODUCCION

Las caídas en adultos mayores son un problema común que puede causar lesiones graves. En muchos casos, cuando una persona sufre una caída no puede pedir ayuda inmediatamente, lo que puede empeorar la situación.

Debido a esto, resulta útil desarrollar dispositivos que puedan detectar este tipo de eventos y generar una alerta. En este proyecto se propone el desarrollo de un sistema electrónico capaz de identificar posibles caídas utilizando un sensor de movimiento.

El sistema analizará la información del sensor para detectar movimientos bruscos o cambios fuertes en la posición. Cuando se detecte una posible caída, el dispositivo activará una alerta para indicar que ocurrió el evento.

Este proyecto busca aplicar conceptos básicos de sistemas electrónicos y programación para desarrollar un prototipo funcional con un enfoque biomédico.

## DESCRIPCION DEL PROBLEMA

Las caídas representan una de las principales causas de lesiones en adultos mayores. Cuando una persona se cae, puede quedar en una situación vulnerable si no recibe ayuda rápidamente.

Actualmente existen algunos dispositivos que ayudan a detectar caídas, pero muchos de ellos son costosos o requieren equipos adicionales. Por esta razón, resulta interesante explorar soluciones más simples que utilicen sensores de movimiento y sistemas electrónicos.

El problema que se busca abordar en este proyecto es cómo detectar automáticamente una posible caída utilizando un dispositivo electrónico que pueda reconocer movimientos fuertes o cambios bruscos en la posición del cuerpo.


## OBJETIVO DEL PROYECTO

# OBJETIVO GENERAL

Diseñar e implementar un sistema electrónico capaz de detectar posibles caídas mediante el uso de un sensor de movimiento y generar una alerta cuando se identifique un evento de este tipo.

# OBJETIVOS ESPECIFICOS

-Medir el movimiento del dispositivo utilizando un sensor.
-Analizar los cambios en el movimiento para identificar posibles caídas.
-Activar una alerta cuando se detecte un evento sospechoso.


## Alcance del proyecto

El proyecto consiste en desarrollar un prototipo funcional que permita detectar posibles caídas utilizando un sensor de movimiento. El sistema contará con un dispositivo electrónico encargado de leer la información del sensor y analizar los cambios en el movimiento.

Cuando el sistema detecte un evento que pueda corresponder a una caída, se activará una alerta sonora o visual.
